O que é html

(Voltar)

HiperText Markup Language, ou simplesmente HTML, é a linguagem básica da web. Com ela é possivel criar sites completos, adicionando titulos, textos, mídia, tabelas, e etc.
Ao contrário do que se pensa, HTML não é uma linguagem de programação, mas sim de marcação, que utiliza-se de tags para delimitar elementos.
De forma simples, as tags são como se fossem pontuações do Português, que nos ajuda a compreender o sentido que o autor quis passar. Dessa forma é preciso conhecer bem essas tags para desenvolver sites que sejam visualmente bonitos e eficientes.
A forma correta de usar as tags são colocando o nome da tag entre os simbolos de maior e menor no início do elemento, e uma barra com o nome da tag no fim do elemento, entre os simbolos de maior e menor.
Elemento é todo e qualquer conteúdo que se queira inserir na pagina, como um símples parágrafo, uma imagem, uma tabela...
Abaixo uma pequena demonstração de um código HTML

<p>Isto é um parágrafo</p>

Tags básicas

(Voltar)

Tags h1 a h6

Tags h1 a h6 são tags título. Com elas um determinado trecho de texto adquire formatação e destaque diferente dos demais. Ex:

Titulo 1

Titulo 2

Titulo 3

Titulo 4

Titulo 5
Titulo 6

Tag p

A tag p, como já foi mostrado acima, é uma tag que comporta um paragrafo, ou seja, um trecho de texto com formatação e destaque diferente das tags h1 a h6. Esta pequena explicação é um exemplo do uso da tag p


Tag mark

A tag mark serve para destacar um trecho de código, muito parecido com um marcador. Ex:

Este é um trecho qualquer, mas esta parte merece destaque visual.


Tag small

A tag small serve para diminuirmos um trecho de texto. Ex:

Esta parte está em tamanho padrão, mas esta parte deve ser menor.


Tag i

A tag i destaca um trecho de texto para itálico.


Tag u

a tag u determina que um determinado trecho deve ficar sublinhado.


Tag strong

A tag strong dá outro tipo de destaque para um trecho de texto. Com isso uma palavra forte receberia uma formatação em negrito.


Tag ol, ul e suas filhas li

As tags ol e ul são tags para listagem de algo. A diferença entre elas é que uma é numéricamente ordenada (ol) e outra não. Ambas recebem uma lista de itens (li) em seu interior. Ex:

    Lista Ordenada

  1. Item 1
  2. Item 2
  3. Item 3

    Lista não ordenada

  • Item 1
  • Item 2
  • Item 3

Tag a

A tag a, ou tag âncora, é utilizada quando se quer criar uma área clicável. É possível acessar conteúdos internos da mesma página, conteúdos internos do mesmo site ou conteúdos externos, de outros sites. Ex:

Conteúdo interno da mesma página (Titulo Principal)
Conteúdo interno do mesmo site (Outra página do site)
Conteúdo externo (Formação em Front-end DIO)

Tag hr

A tag hr cria esta linha horizontal


Tags sup e sub

As tags sub e sub servem para adicionar algum conteúdo acima ou abaixo da linha básica de um elemento. Ex:
Para potências: 102
Para fórmulas químicas: H2O

Algumas Recomendações

(Voltar)

Para o desenvolvimento do conhecimento em HTMl é importante que outras tag sejam estudadas,
como as section, div, nav, aside, e assim por diante.

Todas essas tags, e muitas outras, podem ser estudadas a partir da Documentação oficial da MDN web Documents e da w3schools.

Para além do HTML, é importante conhecer também o CSS, que é uma linguagem de estilos, que complementa o HTML,
dando um pouco mais de beleza à sua página web, e JavaScrip, que é uma linguagem de programação web que adiciona funcionalidades e dinamicidade as suas páginas, deixando-as mais atrativas e divertidas.

Com o domínio dessas três linguagens você pode se tornar um profissional de Front-end e dar continuidade aos estudos, aprimorando suas páginas com conhecimentos de UI/UX.
